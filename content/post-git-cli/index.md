## 내가 자주쓰는 git 명령어 들

초기화
```shell
git init
```
<br>


저장소 추가
```shell
git remote add origin http://나의깃.주소.주소.git
```
<br>


해당 브랜치 pull
```shell
git pull origin master
```
<br>


해당 브랜치 checkout
```shell
git checkout stage
```
<br>


변경 된 내용 걍 되돌리기
```shell
git reset --hard
```
<br>


변경 된 특정 파일만 되돌리기
```shell
git checkout -- 파일이름
```
<br>
